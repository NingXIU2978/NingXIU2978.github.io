Git 如何初始化并提交仓库？

  1. 检查是否已初始化：查看是否存在 .git 文件夹。

    • 有 → 可直接使用 Git 命令管理；
    • 无 → 需执行 git init 初始化仓库。

  2. 添加文件到暂存区：使用 git add <文件名> 或 git add . 添加所有文件。

  3. 提交到本地仓库：使用 git commit -m "提交描述" 提交修改。

  4. （可选）关联远程仓库并推送：
    git remote add origin <远程地址>
    git push -u origin <分支名>

📌 核心命令三步走：
git init → git add . → git commit -m "xxx"
